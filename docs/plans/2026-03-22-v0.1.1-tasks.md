## 1. 状态约定

| 状态 | 含义 |
|------|------|
| `待开始` | 未动工 |
| `进行中` | 正在实现（可同时只有少量任务为进行中） |
| `已完成` | 代码/文档已落地且自测通过 |
| `阻塞` | 依赖外部或需决策，暂停 |

**维护约定（执行 Agent）**：每完成一项，将状态改为 `已完成`，并在 [§3 变更记录](#3-变更记录) 追加一行（日期、任务 ID、简述）。若任务拆分或合并，同步改本表并注明原因。
