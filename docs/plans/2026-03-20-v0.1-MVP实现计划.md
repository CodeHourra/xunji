# 寻迹 v0.1 MVP 实现计划

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 实现寻迹最小闭环——手动同步 Claude Code / Cursor 对话，展示会话列表，单条 LLM 分析生成知识笔记，支持搜索和笔记查看。

**Tech Stack:** Tauri 2.0, Rust (rusqlite, serde, notify, tokio), Vue 3 + TypeScript, Radix Vue, UnoCSS, Pinia, Bun, OpenAI-compatible API

```rust
pub struct Database {
    conn: rusqlite::Connection,
}

impl Database {
    pub fn open(path: &Path) -> Result<Self> {
        fs::create_dir_all(path.parent().unwrap())?;
        let conn = Connection::open(path)?;
        conn.execute_batch("PRAGMA journal_mode=WAL; PRAGMA foreign_keys=ON;")?;
        let db = Self { conn };
        db.migrate()?;
        Ok(db)
    }
}
```

**Step 2: 实现 Claude Code 解析器**

`collector/claude_code.rs`：

关键解析逻辑：
```rust
fn parse_jsonl(path: &Path) -> Result<Vec<NormalizedMessage>> {
    let file = File::open(path)?;
    let reader = BufReader::new(file);
    let mut messages = vec![];

    for line in reader.lines() {
        let entry: serde_json::Value = serde_json::from_str(&line?)?;
        let msg_type = entry["type"].as_str().unwrap_or("");
        if msg_type != "user" && msg_type != "assistant" { continue; }

        let message = &entry["message"];
        let content = extract_content(message);
        if content.is_empty() { continue; }

        messages.push(NormalizedMessage {
            role: message["role"].as_str().unwrap_or("").to_string(),
            content,
            timestamp: entry["timestamp"].as_str().map(String::from),
            tokens_in: entry["usage"]["input_tokens"].as_u64().unwrap_or(0) as u32,
            tokens_out: entry["usage"]["output_tokens"].as_u64().unwrap_or(0) as u32,
        });
    }
    Ok(messages)
}
```

```rust
fn extract_lexical_text(node: &serde_json::Value) -> String {
    match node["type"].as_str() {
        Some("text") => node["text"].as_str().unwrap_or("").to_string(),
        Some("code") => format!("```\n{}\n```", node["code"].as_str().unwrap_or("")),
        _ => {
            if let Some(children) = node["children"].as_array() {
                children.iter().map(extract_lexical_text).collect::<Vec<_>>().join("")
            } else {
                String::new()
            }
        }
    }
}
```

**Step 2: 实现 API Provider**

`distiller/api-provider.ts`：
- 使用 `openai` npm 包（兼容所有 OpenAI-compatible API）
- 构造函数接收 `{ baseUrl, apiKey, model }`
- `distill(prompt, content)` → 调用 chat completions API → 返回文本

```typescript
import OpenAI from 'openai';

export class ApiProvider {
  private client: OpenAI;
  private model: string;

  constructor(config: { baseUrl: string; apiKey: string; model: string }) {
    this.client = new OpenAI({ baseURL: config.baseUrl, apiKey: config.apiKey });
    this.model = config.model;
  }

  async distill(prompt: string, content: string): Promise<string> {
    const response = await this.client.chat.completions.create({
      model: this.model,
      messages: [
        { role: 'system', content: prompt },
        { role: 'user', content },
      ],
      temperature: 0.3,
    });
    return response.choices[0]?.message?.content ?? '';
  }
}
```
